# Sistema de color real (OKLCH)

Los tokens de este proyecto estan definidos en `src/index.css` usando valores `oklch(...)`.  
En Storybook deben usarse con `var(--token)` para reflejar exactamente el tema actual.

## Reglas de uso

- Usa tokens semanticos, no valores hardcoded (`#fff`, `#111`), para que claro/oscuro funcione automaticamente.
- `primary` solo para accion principal.
- `muted` para informacion secundaria, nunca para CTA.
- `ring` siempre visible en foco de teclado.
