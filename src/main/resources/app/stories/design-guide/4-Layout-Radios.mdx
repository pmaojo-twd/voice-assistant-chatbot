# Tokens de estructura

## Radios reales del sistema

Los radios dependen de `--radius` en `src/index.css`.

- `--radius-sm`: `calc(var(--radius) - 4px)`
- `--radius-md`: `calc(var(--radius) - 2px)`
- `--radius-lg`: `var(--radius)`
- `--radius-xl`: `calc(var(--radius) + 4px)`
- `--radius-2xl`: `calc(var(--radius) * 1.8)`

## Regla de consistencia

- Inputs y botones: `rounded-md`
- Tarjetas inline: `rounded-xl` o `rounded-2xl`
- Superficies fullscreen: `rounded-2xl` en paneles internos

## Densidad para chat

- Espaciado vertical entre bloques: `8px` a `16px`
- Padding interno de card inline: `12px` a `16px`
- Distancia minima entre CTA y texto: `8px`
