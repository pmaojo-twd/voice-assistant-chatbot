# Espaciados

La escala de spacing del proyecto es la default de Tailwind v4 (1 unidad = `0.25rem` = `4px`).

## Escala visible

    { token: "space-2", value: "8px", w: "8px" },
    { token: "space-3", value: "12px", w: "12px" },
    { token: "space-4", value: "16px", w: "16px" },

        }}

## Reglas de uso

- **Densidad chat / inline cards:** `gap-2` a `gap-4`.
- **Padding interno controles:** `px-3 py-2` (default `Button`/`Input`).
- **Padding interno cards:** `p-4` a `p-6`.
- **Separación entre secciones de página:** `space-y-8` o `space-y-12`.
- **Margen de respiración alrededor de CTA principal:** mínimo `space-2` (8px).

## Anti-patrones

- Evita valores hardcoded (`margin: 13px`). Si necesitas un valor entre dos pasos, prefiere el más generoso.
- No uses spacings menores que `space-1` (4px) salvo en componentes muy compactos.
