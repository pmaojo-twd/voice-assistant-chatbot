# Tripilots design tokens

Estos tokens son la base troncal del Storybook de Tripilots. Documentan la direccion visual pedida en el roadmap y quedan separados del tema actual de la app para no romper la UI productiva mientras construimos el design system.

## Color palette

{tripilotsColorGroups.map((group) => (
  <div key={group.name}>
    <h2>{group.name}</h2>
    <ColorPalette>
      {Object.entries(group.values).map(([name, value]) => (
        <ColorItem
          key={`${group.name}-${name}`}
          title={name}
          subtitle={value}
          colors={{ Base: value }}
        />
      ))}

  </div>
))}

## Usage rules

- `navy` define surfaces premium, navegacion y texto de alta jerarquia.
- `gold` solo para acento de lujo, estados premium y CTA muy medidos.
- `sand` sirve de fondo calido, no de color de accion.
- `status` se mantiene semantico y no se reutiliza como branding.
- `luxury` shadow se reserva para highlights, no para todo el sistema.
